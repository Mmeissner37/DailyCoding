Introduction

What is Python?

Python is a popular programming language created by Guido van Rossum. It was released in 1991.

It is used for:

  • Server-side web development
  • Software development
  • Mathmatics
  • System scripting

    • What can it do?

  • Used on a server to create web applications
  • Used alongside software to create workflows
  • Can connect to database systems, and read and modify files
  • Used to handle big data and perform complex mathematics
  • Used for rapid prototyping, or for production-ready software development

    • Installing

    Many computers already have Python installed.

    To check on a windows computer, run the following command using the Command Line (cmd.exe):

    c:\Users\Your Name>python --version
